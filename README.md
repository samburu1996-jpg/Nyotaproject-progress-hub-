## 🚀 Getting Started

### View Locally
1. Clone the repository:
   ```bash
   git clone https://github.com/samburu1996-jpg/Nyotaproject-progress-hub-.git
   cd Nyotaproject-progress-hub-
   ```

2. Open `index.html` in your browser or use a local server:
   ```bash
   python -m http.server 8000
   # Then visit: http://localhost:8000
   ```

### Make Changes
1. Edit `index.html`, `styles.css`, or `script.js`
2. Commit and push to GitHub:
   ```bash
   git add .
   git commit -m "Your message here"
   git push origin main
   ```
3. Changes will automatically appear on your GitHub Pages site!

## 💡 How to Customize

### Update Content
Edit `index.html` to:
- Change project title and description
- Update progress percentages
- Add team member information
- Modify contact details

### Change Colors
Edit the `:root` section in `styles.css`:
```css
:root {
    --primary-color: #667eea;      /* Change this */
    --secondary-color: #764ba2;    /* Change this */
    --accent-color: #f093fb;       /* Change this */
}
```

### Add More Sections
Add new `<section>` elements in `index.html` following the same structure.
